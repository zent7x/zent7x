### What I'm building

**[mcp-guard](https://github.com/zent7x/mcp-guard)** — a security & network MCP server for Claude Code and Cursor, written in Go.
22 tools for secret scanning, CVE checks, TLS/DNS/header audits, and local network + forensics work. Most tools need a physical machine to run, so they do things a cloud model simply can't.
